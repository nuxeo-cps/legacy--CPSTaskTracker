 - Unpack the tarball in your $ZOPE_HOME/Products directory.

   $ cd $ZOPE_HOME/Products 
   $ tar -xzf CPSTaskTracker-X.X-X 

 - Restart Zope

 - Through ZMI, go at the root of CPS and create an external method as stated below. 

   Id : CPSTaskTracker Installer (or whatever)
   Name : whatever
   Module Name : CPSTaskTracker.install
   Function Name : install

 - Save it 

 - Click on the just created external method 'CPSTaskTracker Installer' 

 - Click the test tab.
 
 - That's it.
